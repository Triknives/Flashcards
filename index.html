Flashcards For Learnin Good

Variables

Variables can be thought of as containers used to store information. They allow for a way to label data with a descriptive name.

Strings

Strings a JavaScript data type that represents the exact text of whatever is enclosed in the quotes.

Methods

Methods A method is an action run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response.

Operators

Operators An operator is a special character (or characters) that indicates an action to be performed. I.E: , -, *, /, and %
